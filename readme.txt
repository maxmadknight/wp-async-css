== Description ==
This plugin will hook into the WordPress style handling system and loads the selected stylesheets asynchronous.

== Installation ==
1. Copy the `wp-async-css` folder into your `wp-content/plugins` folder
2. Activate the `WP Async CSS` plugin via the plugins admin page
3. Navigate to Settings and `WP Async CSS` to select which stylesheets you want to load asynchronous.
